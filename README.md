# pb_containerized

Containerized Pockebook SDK that can be used on any OS where docker/podman is available. [The similar project](https://github.com/Skeeve/SDK_6.3.0) which this one is inspired by doesn't work any more as it uses old Debian version and svn, which is [no longer supported by Github since 2024](https://github.blog/news-insights/product-news/sunsetting-subversion-support/).
